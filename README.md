# 🚖 Shree Tours & Travels

Shree Tours & Travels is a full-stack taxi booking web application developed to simplify the online cab booking process. Users can search available vehicles, complete bookings using OTP-based mobile verification, download booking confirmation PDFs, and view their booking history.
----------------------------------------------------------------------------------------------------------------------------
Features
🚖 Search available vehicles based on route, date, and time
📱 OTP-based mobile number verification
👤 Automatic registration for new users
✅ Booking confirmation with a unique Booking ID
📄 Download booking confirmation as PDF
📚 Booking history management
🚗 Vehicle availability management
📱 Responsive and user-friendly interface
